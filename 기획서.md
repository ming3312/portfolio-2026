# 포트폴리오 기획서 (V3.1 - 최종 검수본)

본 기획서는 화면 노출 텍스트에서 구체적인 프로그램/라이브러리명("React", "Next.js", "GSAP", "Three.js", "WebGL", "Node.js", "MongoDB", "Vercel" 등) 및 구체적인 수치("1px", "100%" 등)를 완전히 배제하고, 범용적인 기술 개념(Branding, Interaction, 3D Web Graphics, API System 등)으로 대체한 최종 튜닝 가이드라인을 반영하여 작성되었습니다.

---

## 1. 포트폴리오 개요

*   **목적**: 경력 이직 (웹 디자이너 & 퍼블리셔) + 프리랜서 외주 수주 및 비즈니스 홍보
*   **핵심 포지션**: 웹 디자이너 및 퍼블리셔 (디자인과 코딩을 아우르는 올라운더 웹 크리에이터)
*   **서브 역량**: 생성형 인공지능 API를 결합한 유체 풀스택 시스템 아키텍처 및 클라우드 배포 구축
*   **비주얼 컨셉**: **Hyper-Interactive Cyber Liquid (하이퍼 인터랙티브 사이버 리퀴드)**
*   **컬러 전략**: Matte Obsidian Black BG + Vivid Hot Pink 메인 (9%) + Neon Yellow 서브 (1%)

---

## 2. 메인 브랜드 슬로건

> **"감각적인 픽셀에서 단단한 서버까지. AI를 융합한 유연한 빌딩으로 최상의 웹 경험을 빠르게 구체화합니다."**

*   **의도**: 디자이너의 감각(픽셀)과 개발자의 논리적 설계(서버)를 대치하며, 다양한 인공지능 협업 기술을 융합하여 한층 더 다채로워진 구현 스펙트럼과 효율적이고 신속한 웹 구현 속도를 강력하게 전달합니다.

*   **Project 01: HWANGWOO BRANDING**

    *   *설명*: 정교한 미니멀 라인과 세련된 다크 테마를 결합한 브랜드 쇼룸 구현.
    *   *기술 태그*: `Branding`, `Interaction`, `Layout`, `Publishing`
*   **Project 02: WORLD VISION CAMPAIGN A**
    *   *역할*: `CREATIVE DIRECTING & BUILD`
    *   *설명*: 월드비전 긴급 구호 캠페인 마이크로 웹사이트. 사용자 조작과 휠 스크롤 속도에 매끄럽게 매핑되는 액체 왜곡 모션 연동.
    *   *기술 태그*: `Motion Graphics`, `3D Graphics`, `Interactive Web`, `Design`
*   **Project 03: WORLD VISION CAMPAIGN B**
    *   *역할*: `INTERACTIVE GRAPHICS & PUBLISH`

    *   *기술 태그*: `Interactive UI`, `UX Logic`, `Smooth Motion`, `Publishing`

### 4) Services (외주 제작 가이드)
얇은 네온 실선과 아코디언 컴포넌트를 사용해 프리랜서 비즈니스 포트폴리오로서 신뢰성 있는 서비스 라인업을 구체화합니다.

*   **Service 01: BRANDING & INTERACTION DESIGN**
    *   *설명*: 감각적인 브랜드 시각 언어(Identity) 수립 및 웹의 물리적 깊이를 표현하는 마이크로 인터랙션 기획.
    *   *세부 태그*: `UI/UX Strategy`, `Micro Interaction`, `Obsidian Theme Design`, `Grid Identity Layout`
*   **Service 02: HIGH-END WEB PUBLISHING**
    *   *설명*: 비주얼 설계를 기반으로 오차 없는 정교한 웹 크리에이션. 마우스 스크롤의 관성 전환과 유체 역학 모션 배경을 적용한 퍼블리싱 완수.
    *   *세부 태그*: `Interactive Motion Design`, `3D Web Graphics`, `Custom Fragment Shaders`, `Responsive Fluid Layout`
*   **Service 03: AI INTEGRATION & FULL-STACK SYSTEM**
    *   *설명*: 최신 대화형 인공지능 기술 연동 및 지능형 검색 엔진 파이프라인 설계. 커스텀 백엔드 서버 빌딩과 데이터베이스 아키텍처 구축, 자동화 서버 시스템 및 신속한 클라우드 배포 완성.
    *   *세부 태그*: `AI Technology Integration`, `Custom API Backend System`, `Database Architecture Design`, `Cloud Server Deployment`
